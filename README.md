# wifi_boss

Impleted a crossplatform application using flutter, that can be used to login the wifi.

## advanced features
- added dark thememode
- added save credentials feature
- added fun emoji aknowledgement
## packages used
- http

- url_launcher

- sqflite

- font_awesome_flutter
